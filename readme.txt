# Forged-Server

## Start
Launch `Start-Server_UNIX.sh` for unix systems or `Start-Server_WINDOWS.bat` for windows systems.

## Configuration Client
Download all [mods](https://github.com/Rubik91/MinecraftServer_Forge/raw/server-20170225/Server/mods/jei_1.12.2-4.8.5.147.jar)

### Windows

1. Press Win + r.
2. Type in %appdata%\.minecraft.
3. Open or create the `mod` folder.
4. Paste all mods in the `mod` folder.

### Linux

1. From Nautilus (or most other file browsers), press `ctrl + H` to see hidden folders the navigate to `.minecraft`.
   If in the console, type, `ls -a ~/` and you will find the `.minecraft` directory.
3. Open or create the `mod` folder.
4. Paste all mods in the `mod` folder.
